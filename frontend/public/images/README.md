## Logo

- **Fichier attendu** : `logo.png` (ou `logo.svg`)
- Le logo s’affiche dans l’en-tête du site à côté du nom « FAB ».
- Formats conseillés : PNG avec fond transparent, ou SVG. Taille conseillée : environ 32–40 px de hauteur pour l’affichage dans le header.
